ARM Training

Here are two courses written to introduce my former colleagues at Picsel Technologies to the joys of programming in ARM assembly language.

Introduction to ARM
The first course introduces the reader to ARM assembly language. It goes on to explain the development of the ARM architecture and the capabilities of the various ARM CPU cores.
After reading this you should be able to understand ARM code well enough to understand the examples in the next course...
Efficient C for ARM
The second course is about tuning. It looks at how we can produce efficient ARM code by altering the way we write our C code.

The courses were presented to over 70 engineers in Glasgow, Edinburgh, Reading, Malaysia and Korea.

When delivered, each course would usually take three hours to run through. The course would be presented sandwich style: two lecture parts and a delicious centre filling where the student gets to experiment with some small routines.

Caveats

The courses were manually converted across from the PowerPoint slides and separate trainer's notes I used when presenting the courses. This may show in the terseness of some of the pages. Some detail is sacrificed for the sake of making a point.

Some closely related sections have been merged together, some were left separate. Some pages are of substantially different lengths.

Over time I'll rework the content to better suit its new format.

The courses were written for an environment where ARM Developer Suite (ADS) was available, so all of the assembler examples are in ADS format. If you intend to try the examples on a more affordable compiler, such as GCC, you will need to use GNU ARM assembler syntax.

Assembler examples were compiled with ADS 1.2's armcc and GCC 2.95. You will find that other compilers produce different code, of varying quality.
