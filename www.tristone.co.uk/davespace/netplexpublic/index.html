Netplex

    Netplex is a flexible and easy to use World-Wide Web server for RISC OS 3.1 or later. To get it up and running requires no editing of configuration files - you simply drag the directory containing your Web pages to the configuration window, save the configuration and then restart.

    Netplex is (optionally) compliant to HTTP/1.1. Full support is included for CGI programs using the GET and POST methods. The maximum number of connections and overall transfer rates can be adjusted as can be the port upon which the server operates, allowing for great flexibility in determining how much of your network bandwidth is used by the server.

    To function, Netplex requires a standard Internet TCP/IP stack, so it will not currently work with Doggysoft's Termite Internet package.

    Netplex is Shareware. Registered users of Netplex are entitled to receive newer versions and are given access to the private pages on this server. Registration costs £5 (five pounds Sterling).
